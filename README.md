https://docs.pushwoosh.com/platform-docs/pushwoosh-sdk/android-push-notifications

Maven integration:

	<dependency>
  		<groupId>com.pushwoosh</groupId>
  		<artifactId>pushwoosh</artifactId>
  		<version>6.7.46</version>
	</dependency>
